04. Frontend Basics

1주 차 과제: 프론트 기본 개념 정리

1. display 속성 - Block: 한 줄 차지 (div)
- Inline: 컨텐츠만큼 차지 (span)
2. HTML Layout - Flex: 1차원 정렬
- Grid: 2차원 정렬
- Float: 요소 부유 설정
3. 박스 모델 - Content: 실제 내용 영역
- Padding/Border/Margin: 안팎 여백
4. id & class - id: 고유한 식별자 (#)
- class: 중복 가능한 그룹 (.)
