## 🛠️ 개발 환경 (Dev Environment)
- **IDE:** VS Code 
- **Version Control:** Git / GitHub

## 📂 프로젝트 구성 (File Structure)
본 프로젝트는 단일 페이지를 넘어 실제 웹 애플리케이션의 구조를 연습하기 위해 다중 페이지 및 모듈 분리 형태로 구성했습니다.

- `index.html`: 메인 대시보드 및 네비게이션 허브
- `intro.html`: 자바스크립트를 활용한 슬라이드(Carousel) 형태의 자기소개
- `style.css`: 공통 UI 디자인 및 슬라이드 애니메이션 스타일시트
- `script.js`: 슬라이드 전환 및 인디케이터 제어 로직
